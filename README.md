# To Do List app
- Use vanilla HTML, CSS, and JavaScript. to do a small To Do List

## Features
- **Add tasks** — type and press Enter or click Add
- **Complete tasks** — check the checkbox to mark as done
- **Edit tasks** — click the edit button to rename a task
- **Delete tasks** — remove tasks individually
- **Drag and drop** — reorder tasks by dragging
- **Filter tasks** — view All, Active, or Completed tasks
- **Task counter** — live count of total, active, and completed tasks
- **Duplicate warning** — warns if a task already exists, with option to add anyway
- **Dark mode** — toggle between light and dark theme, persists on reload
- **Persistent storage** — all tasks and theme saved to localStorage

## Getting Started
1. Clone or download the repository
2. Open `index.html` in your browser
3. No build step or dependencies required
